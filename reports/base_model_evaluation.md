# Base Model Evaluation Report

This report evaluates the performance of the base model (**Qwen2.5-1.5B**) on 10 domain-specific IT Helpdesk questions. Since the base model has not been fine-tuned on corporate IT policies or IT helpdesk structures, its responses tend to be generic, lack specificity, and fail to refer to internal company procedures.

## Evaluation Summary

| # | Question / Query | Base Model Answer | Problem identified |
|---|------------------|-------------------|--------------------|
| 1 | **Password Expiry / Reset**<br>Subject: Password reset request.<br>Query: I cannot log in, password expired. How can I reset it? | Please contact your system administrator or IT helpdesk department to request a password reset. You may need to provide your username and identification. | **Too Generic**: Does not provide details about self-service portals, password requirements (e.g. 14 chars), or the 90-day expiry rule. |
| 2 | **VPN Access Issue**<br>Subject: VPN connectivity issue.<br>Query: I am trying to connect to the VPN from home using Cisco AnyConnect but it keeps saying connection failed. How can I fix this? | Check your internet connection. Make sure you are entering the correct VPN address and credentials. If it still fails, contact IT support. | **Incomplete Troubleshooting**: Fails to mention corporate requirements like using Microsoft Authenticator for MFA, or the 8-hour auto-timeout. |
| 3 | **Software Request**<br>Subject: Software Installation.<br>Query: I need to install IntelliJ IDEA on my work laptop for development. How can I do that? | You can download IntelliJ IDEA from JetBrains website. Run the installer and follow the instructions to set it up. | **Policy Violation**: Instructs the user to download software directly from the web, violating standard corporate policy (should direct to Software Center / Self Service). |
| 4 | **Slow Laptop / Hardware**<br>Subject: Laptop running very slow.<br>Query: My Lenovo ThinkPad T14 is lagging and freezing constantly. Can I request a hardware upgrade or RAM increase? | You should clean up your hard drive, close unused programs, or check for malware. If it is still slow, ask your manager if you can buy a new computer. | **Unprofessional / Lack of Policy**: Fails to reference the 3-year replacement cycle, the RAM upgrade request process (screenshot of Task Manager showing >85% usage), or procurement forms. |
| 5 | **Guest WiFi Access**<br>Subject: WiFi Access.<br>Query: My client is visiting the office today. How can they connect to the guest Wi-Fi? | You should check if there is an open Wi-Fi network in the office. Otherwise, ask the receptionist for the Wi-Fi password. | **Security Risk**: Suggests looking for random open Wi-Fi networks. Fails to mention the official "Company-Guest" network, its 24-hour expiration, and lack of internal system access. |
| 6 | **Data Sync / OneDrive**<br>Subject: Data Backup.<br>Query: I lost my local files when my laptop crashed. Can you restore them? | If you did not make a backup of your files, they might be lost forever. You can try running file recovery software or taking it to a repair shop. | **Helpdesk Failure**: Fails to ask if they synced files to OneDrive for Business (standard backup policy) and does not offer standard system restoration assistance. |
| 7 | **Phishing Attempt**<br>Subject: Suspicious Email.<br>Query: I received an email asking me to click a link to verify my HR benefits. It looks suspicious. What should I do? | If you think it is fake, you can delete it or ask your colleagues if they got it too. Don't click on any link. | **Inadequate Security Response**: Fails to instruct the user to use the 'Report Phishing' button in Outlook or notify the SOC immediately. Suggesting "asking colleagues" is bad practice. |
| 8 | **MFA Token Loss**<br>Subject: MFA Reset.<br>Query: I got a new phone and lost access to Microsoft Authenticator. How can I reset my MFA? | Go to your account security settings and click reset. Or ask your admin to disable MFA on your account. | **Security/Vulnerability**: Fails to outline the identity verification process (video call / photo ID check) required before generating temporary bypass codes. |
| 9 | **Printer Mapping**<br>Subject: Printer setup.<br>Query: How do I add the 3rd floor office printer to my laptop? | Go to settings, printers, and click add printer. Select the printer from the list of nearby devices. | **Lack of Internal Detail**: Doesn't mention mapping via the print server (\\\\printserver\\printer_name) or standard Active Directory printer directory mapping. |
| 10| **Device Offboarding**<br>Subject: Asset Return.<br>Query: Today is my last day. What is the process for returning my laptop and monitor? | You should pack them up and leave them on your desk or ask your supervisor where to put them. | **Vague**: Does not state the 5-day return deadline, returning items directly to IT, or details on security wipes. |

## Conclusion

The base model lacks specific corporate IT policy awareness and structures. It frequently gives generic web-search-style advice or recommends checking with external services. SFT (Supervised Fine-Tuning) is required to feed the model the specific IT policy guidelines and response structures.
